打造一个实时陪伴型 AI 桌面宠物，为用户提供情绪价值，并完成简单的任务
- 被动聊天：支持桌面聊天 AI 交互
- 主动聊天：后台长时运行，监测用户行为，主动寻找话题与用户聊天
- 自我进化：自己有独立的情绪、记忆、技能，随着交互过程加强对用户的了解
- 通用任务：完成简单的定制化任务。
- 多模态支持：要支持图片写入与图片生成

约束条件：
- UI 要美观可爱。
- 代码要模块化、稳定可靠、易维护。
- 开发时要以生产级别的代码要求自己，不要写出 Demo 一样粗糙不可维护无法阅读的代码。
- 不需要考虑隐私过滤相关的东西。
- 开发时不要写没用的测试代码。保证的你的测试真的有意义。
- 不要考虑云同步等需求。
- 不要考虑专注模式相关的需求。
- 不做深色 / 浅色主题
