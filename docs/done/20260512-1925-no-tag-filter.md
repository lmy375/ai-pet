# PanelTasks tag filter "🚫 无 tag" 选项

## 需求

tag 过滤行只列出 tasks 中出现过的 tag。当用户想找"我还没打 tag 的
任务"（整理时优先处理 / 流程未完整的草稿）—— 没有入口。补一个合成
chip，与 tag chips 共栏。

## 实现

`src/components/panel/PanelTasks.tsx` 三处改：

1. **filter 链 (line ~1671)**：在 selectedTags.size > 0 的分支加一条
   特例 —— 若 `selectedTags.has("")`（空串 sentinel）且 `t.tags.length === 0`
   命中，让该任务通过。与多选 tag 是 OR 关系（同时选"无 tag" + 真实
   tag → 看并集），与多选普通 tag 的语义保持一致。
   - 空串作 sentinel 是安全的：parseTag 拒绝空 → 真实 tag 不会是 ""

2. **derive `untaggedCount` (line ~1789)**：useMemo `tasks.filter(t =>
   t.tags.length === 0).length`。仅 > 0 时才浮 chip —— 全部任务都打了
   tag 时这 chip 是噪声。

3. **render (line ~2877)**：
   - 把外层条件从 `allTags.length > 0` 扩到 `... || untaggedCount > 0`
     —— 兼容"全部任务都未打 tag"的边界（旧条件下整行都隐藏，"无 tag"
     chip 无处显）
   - tag chip 列表后插 IIFE：当 untaggedCount > 0 渲合成 chip
     - 文案 "🚫 无 tag (N)"
     - `borderStyle: "dashed"` —— 视觉区分它是"谓词类别"而非普通 tag
     - 复用 toggleTag("") + s.tagFilterChip(selected) 同 style 路径
     - hover tooltip 显当前 untagged 计数 + 行为提示

## 验证

- `npx tsc --noEmit`：clean
- 行为：
  - 全部任务都打了 tag → 不显 "🚫 无 tag" chip
  - 有 N 条任务未打 tag → 行末显 dashed chip "🚫 无 tag (N)"
  - 点击 → selectedTags 加入 ""，仅未打 tag 任务可见
  - 再点 → selectedTags 移除 ""，恢复
  - 同时选 "无 tag" + 真实 tag "工作"  → 既看未打 tag 的，也看带 #工作 的
  - "清除"按钮（既有）一次性清掉所有 selectedTags（含 ""）
  - filtersActive 派生条件复用 selectedTags.size > 0 —— 选 "无 tag"
    一项也会让"✕ 清除全部过滤"按钮 / search row 状态行 / 空状态文案
    都按 filter-active 路径渲染

## 不在本轮范围

- 没把 "无 tag" chip 抽成独立 state（filterNoTag boolean）：复用 ""
  sentinel + 既有 Set + toggleTag 路径只改 3 处；独立 state 要改 filter
  逻辑、filtersActive、clear 按钮、bulk export 等 5+ 处
- 没让 chip 支持"右键改颜色"（现有 tag chip 的菜单）：no-tag 不是
  tag，没颜色语义；右键 noop 自然
- 没做 "🆕 仅今日新建" / "💤 长期未动" 等其它谓词 chip：scope 翻倍；
  当前 dueChip / overdueChip 已覆盖时态维度，本轮只补 tag 维度的"无"
