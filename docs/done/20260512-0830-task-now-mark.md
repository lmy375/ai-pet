# PanelTasks ⚡ NOW 标记 + 桌面 nudge

## 需求

due 字段是绝对时间，"接下来 1 分钟该集中处理这条"没有显式标记。加一个
"⚡ 标 NOW (60s)" 入口：60 秒内该任务浮顶 + 桌面气泡 nudge 一次提醒，
过期自动清除。session 内的"短期专注"信号，不动后端。

## 实现

### `src/components/panel/PanelTasks.tsx`

- 新 state `nowMarkedTitles: Set<string>` + `nowTimersRef: useRef<Map<title, timerId>>`
- 新 `markTaskNow(title)` callback：
  - add title to set
  - `emit("task-now-mark", { title })` 跨窗口通知 pet
  - 清掉同 title 的旧 timer（连续点 = 续计 60s）
  - setTimeout 60s 移除 title from set + 清 timer 记录
- unmount cleanup 清所有 pending timer
- 右键菜单（taskCtxMenu）pending 行加 "⚡ 标 NOW (60s 浮顶 + 桌面 nudge)"
  按钮；已 mark 状态文案改为 "⚡ 续 60s"
- `sortedUnfinished` 末尾追加：mark 集非空时拆 marked / rest 两段拼接，让
  marked 永远在最前（任何 sortMode 下都生效）
- 任务卡标题旁加 "⚡ NOW" 徽章（橙色 tint + 1px border + 脉冲动画 1.6s
  循环），nowMarkedTitles.has(t.title) 时渲染
- inline `<style>` 加 keyframes `pet-task-now-pulse`（scale 1→1.06 + 微
  opacity 变化）+ `prefers-reduced-motion` 退化

### `src/App.tsx`

- 新 useEffect 监听 `task-now-mark` 事件：
  - 立刻 `appendAssistant("✨ 标记「<title>」为 NOW · 1 分钟内 nudge 提醒")`
  - setTimeout 60s 后 `appendAssistant("⏰ 提醒：「<title>」还没动手吗？")`
  - 跟踪 pendingTimers Set，组件 unmount 时清
- 与 PanelTasks 的 60s timer 各自独立，panel 关掉不影响 pet 的 nudge

## 验证

- `npx tsc --noEmit` clean
- 行为：
  - 右键 pending 任务 → 菜单出现 "⚡ 标 NOW (...)" 项
  - 点击 → 任务浮到队列顶 + "⚡ NOW" 橙色脉冲徽章；桌面 ChatMini 立刻收
    到 ack "✨ 标记「<title>」为 NOW · 1 分钟内 nudge 提醒"
  - 60s 后：徽章消失 + 任务回到原排序；桌面气泡再收一条 "⏰ 提醒：「<title>」
    还没动手吗？"
  - 期间再点同条 → 菜单文案变 "⚡ 续 60s"；新计时取代旧
  - 同时 mark 多条 → 都浮顶；徽章脉冲；各自独立 60s
  - 关 panel 重开 → mark 清空（session 内有效）；pet 端 timer 仍跑（独立）

## 不在本轮范围

- 没做"NOW 标记跨重启持久化"：60s 是即时焦点；持久化反而让陈旧 mark 缠
  绕。语义匹配
- 没改后端：纯前端 mark + 局部 nudge；不需要 LLM 介入 / proactive 触发
- 没做 NOW 标记的 chip 过滤入口：列表已浮顶，再加 chip 重复
