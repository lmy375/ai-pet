## 真正的小补：加 hover tooltip

虽然计数已显，但 chip 没有 `title` 属性。hover 时无说明文案，新用户可能
不知道点击会触发过滤。补一行：

```diff
 <span
   key={tag}
   style={s.tagFilterChip(selected)}
   onClick={() => toggleTag(tag)}
+  title={
+    selected
+      ? `点击取消「${tag}」过滤（当前共 ${count} 条带此 tag 的任务）`
+      : `点击只看带「${tag}」的任务（共 ${count} 条）`
+  }
 >
```

跟随 selected 状态描述不同动作；`(N)` 在数字小不易扫的场景下用 tooltip
显式给出 "共 N 条" 文案让用户更确定。
