## 改动

### `src/components/panel/PanelTasks.tsx`

row chip 区（紧贴 🔒 blocked chip 之后、priority button 之前）渲染：

```tsx
{(t.status === "pending" || t.status === "error") && (() => {
  const ts = Date.parse(t.created_at);
  if (Number.isNaN(ts)) return null;
  const ageMs = nowMs - ts;
  if (ageMs < 3 * 86_400_000) return null;  // 3 天阈值
  const rel = formatRelativeAge(t.created_at, nowMs);  // 既有 helper
  if (!rel) return null;
  return (
    <span
      title={`创建于 ${... }（${rel}）—— 放了一阵了，要不要拆 / 改 priority / 取消？`}
      style={{
        background: "color-mix(in srgb, var(--pet-color-muted) 12%, transparent)",
        color: "var(--pet-color-muted)",
        border: "1px solid color-mix(in srgb, var(--pet-color-muted) 25%, transparent)",
        // ... 与既有 chip 同尺寸 / padding / borderRadius
      }}
    >
      🕰 {rel}
    </span>
  );
})()}
```

## 关键设计

- **3 天阈值**：< 3 天的任务"新"，无需提醒（chip 噪音）；≥ 3 天开始显 —— 覆盖 "3 天前 / 一周 / 两周 / 一月" 等粒度，让 owner 看到的都是"放了一阵"的任务。1 天太低（昨天派的也显），7 天太高（5 天的积压看不到）。
- **仅 pending / error 渲**：done / cancelled 是静态终态，年龄无 actionable 信号 —— 任务都结束了，再喊"放了 14 天"是噪音。与既有 💤 snooze chip 终态不渲同思路。
- **muted gray 配色**：与 📌（amber）/ 💤（purple）/ 🔒（yellow）等 actionable chips 配色族错开 —— age 是「信息」非「动作」。background 用 `color-mix(... 12%, transparent)` 在 light/dark 主题下都得到柔和的灰底，不抢视觉。
- **复用 `formatRelativeAge` helper**：既有的 `formatRelativeAgeBuckets` 已覆盖 minute / hour / day 三档；3 天 + 阈值下输出"X 天前"是自然语义。不引入新文案 / 新桶，保字符串 surface 收敛。
- **tooltip 附"要不要拆 / 改 priority / 取消"提示**：把视觉信号转成 actionable 引导，让 owner 知道"看到这个 chip 该做什么"。比单纯显时间多一步价值。
- **嵌在既有 chip 行末尾**：紧贴 🔒 blocked chip 之后、priority button 之前 —— 视觉上属于"任务状态信号"集群，不与 actionable 入口（priority / due picker）混。

## 不做

- **不显"上周 / 上月" 自然语言桶**：TODO 描述提到这两词，但 `formatRelativeAge` 已经是 "N 天前" 数字桶，覆盖一致够用。引入新桶（"本周" / "上周" / "本月" 等）有边界 case 复杂度（"周一 1 点的'上周'是几天？"）而 owner 心智上 "14 天前" 与"两周前"等价 —— 数字桶更精确。
- **不与"任务复盘视图"展开卡片中的 "X 天前创建" 合并**：那个在 detail 展开 + 配合 updated_at 显示活跃度，是 deep-dive；本 chip 是 row-level glance。两层信息层级独立。
- **不和 isRecentlyUpdated 绿点冲突**：绿 ● 表示"最近 updated"（活跃信号），🕰 表示"久 created"（积压信号）。两者完全可以共存：一个老任务刚刚被 owner / LLM 触碰，既绿 ● 又 🕰，正好告诉 owner "这条还在动但起源很久"。
- **不写测试**：纯 UI 时间阈值渲染，无可单测的纯函数；既有 `formatRelativeAge` / `formatRelativeAgeBuckets` 已有覆盖。
